# Examen_CSC
* Description *

Enunciado 
Llevamos un registro de productos de farmacia.

De cada producto sabemos:
- Nombre producto
- Precio
- Precio oferta (si lo tiene)
- Laboratorios
- Cantidad

Registra, al menos, 4 productos y, crea, 2 lógicas: 
- Una que, dado un precio, devuelva productos por valor igual o inferior a ese precio.
- Una lógica que devuelva productos en oferta.

Segundo examen: 
Llevamos un registro de estaciones de metro. De cada estación conocemos:
Nombre
Línea
Año de inauguración
Cantidad de salidas a la calle
Correspondencia con otras líneas o transportes
--------------------------------------------------
Registra, al menos, 4 estaciones y:
Crea una lógica que devuelva las estaciones que NO tengan correspondencias.
Crea otra lógica que, dada una línea de metro, devuelva estaciones
de esa línea.
